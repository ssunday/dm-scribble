# DND Scrawl Client

## Requirements

- Node 14.5.0

## Setup

- `npm install`
- `npm install start`

Visit on [http://localhost:8080](http://localhost:8080)

## Linting

- To see issues: `npm run lint`
- To autofix issues: `npm run lint:fix`

## Testing

- Run: `npm run test`
- Watch: `npm run test:watch`
